# Abiodun Taiwo | Senior Flutter Developer Portfolio

A high-performance, modern professional portfolio built with a focus on "Precision Engineering" and seamless user experience. This project showcases my expertise in architecting complex mobile ecosystems, high-security Fintech solutions, and AI-integrated platforms.

## 🚀 Key Features

- **Glassmorphism UI:** A premium, modern aesthetic using Tailwind CSS with backdrop-blur effects and sophisticated gradients.
- **Dynamic CV Sync:** Integrated with Google Docs to allow resume updates without code changes. The "Download CV" button always serves the latest version as a PDF directly from source.
- **Automated Lead Capture:** Built-in contact form with **Netlify Form Detection**, enabling instant email delivery to Gmail without a backend.
- **Precision Navigation:** Smooth scrolling with precise `scroll-mt` offsets to account for the fixed glass-effect header.
- **Responsive Engineering:** Fully optimized for all screen sizes, from mobile devices to desktop monitors.
- **Performance Focused:** Designed to reflect a "60 FPS Standard" through clean CSS transitions and optimized asset loading.

## 🛠 Tech Stack

- **Framework:** HTML5 / CSS3
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)

## ⚙️ Maintenance & Updates

### Updating the Resume
The resume is dynamically pulled from Google Docs. To update your CV:
1. Edit your content in the linked Google Doc.
2. The site will automatically serve the updated version as a PDF whenever a user clicks "Download CV".
3. **No redeployment required.**

### Contact Form
Submissions are handled automatically by Netlify.
1. Deploy the project to Netlify.
2. Netlify will detect the `data-netlify="true"` attribute.
3. Manage submissions directly in your Netlify dashboard or configure email notifications.
